리액트 JS는 렌더링할 때 전체가 아닌 바뀐 부분만 변경한다.

[ React.useState(초기값) ]
- [초기값,function]을 리턴한다.

const data = React.useState(0);
- data[0] = 0;
- data[1] = function

=> 아래와 같이 표현할 수도 있다.

const [counter,setCounter] = React.useState(0);
- counter = 0;
- setCounter = function;

const onClick = () => {
  // setCounter(3030);
  setCounter((current) => current +1);
}
==> onClick 함수를 실행시키면 setCounter 함수가 실행되어 ()안에 값을 counter에 할당하고 리렌더링한다.
==> 현재 state를 바탕으로 다음 state를 계산하고 싶다면 두번째 방법을 사용한다.
