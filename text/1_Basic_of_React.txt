1. 자바스크립트에서 react를 사용하기 위해서는 두 가지를 import 해야한다. react와 react-dom

[ React.createElement() ] 
- 첫번째 아규먼트는 생성할 태그 이름 
- 두번째 아규먼트는 부여할 프로퍼티를 객체로 담는다. style, id, class
- 세번째 아규먼트는 태그 안에 들어갈 텍스트이다.
- const span = React.createElement("span",{ id:"sexy-span" },"Hello I'm a span");

- 프로퍼티에 이벤트를 추가할 수도 있다. 이때 이벤트는 html에서 사용하는 프로퍼티 이벤트이다. 

 const h3 = React.createElement(
      "h3",
      {
        onMouseEnter: () => console.log("mouseEnter"),
      },
      "Hello I'm a span   "
    );

[ ReactDOM.render() ]
- ReactDOM의 render 메소드는 HTML을 업데이트 시켜준다.
- 첫 번째 아규먼트는 넣을 내용물
- 두 번째 아큐먼트는 넣을 대상
- ReactDOM.render(container,root);
  ReactDOM.render(<Container />, root);


[ JSX ]
- Javascript를 확장한 문법 
- 브라우저가 JSX를 이해할 수 있도록 babel을 설치해야한다. babel standalone
- <태그>내용물</태그>처럼 HTML과 같은 구조를 가진다.
- 컴포넌트의 첫 글자는 대문자여야한다. JSX 방식의 태그를 가리키는 변수

const Button = () => (
  <button style={{ backgroundColor: "tomato" }} onClick={() => console.log("Clicked") }>
   Click me 
  </button>
  );

const Container = (
  <div>
    <Title />
    <Button />
  </div>
)
-- > 이런식으로 태그 안에 다른 태그들을 넣어줄 수 있다.
